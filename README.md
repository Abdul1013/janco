Janco — AI-powered cleaning service platform for African Homes. Built with React Native, FastAPI, Supabase, and OpenCV. 🚀 Clean spaces, happy faces.


---

## 💡 Use Cases

- A tenant books a monthly deep cleaning subscription
- Uploads a photo of their living room
- AI estimates the area, calculates price (₦/m²)
- Janitor is assigned automatically based on location and skill
- Payment is processed securely
- Janitor arrives and updates job status
- Admin tracks job success via dashboard

---

## 📈 Startup Vision

Co-Janitors aims to become Nigeria’s #1 janitorial tech startup, combining logistics, AI, and human labor to clean homes and lives more efficiently. Built by locals, for locals, with global tech standards.
